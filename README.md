## Purpose
DT-Ops simulates operational workflows and security scenarios, enabling risk modeling and AI-driven scenario optimization.


## Features
- Operational scenario simulation
- Digital twin modeling
- Risk assessment dashboards


## Tech Stack
- Python, FastAPI, Streamlit
- Pandas, NumPy, PyTorch
- Dockerized deployment


## Usage
1. Clone repo: `git clone https://github.com/atsuvovor/CyberThreat-Insight.git`
2. Navigate: `cd CyberThreat-Insight/DT-Ops`
3. Run demo: `streamlit run demo_app.py`
